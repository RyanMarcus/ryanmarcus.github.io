Luxer Update! Beta 7

Leave a reply

I released a new version of Luxer today… Beta #7!

Changes:
– Fixed a user monitor bug.
– The say command no longer whispers back to you.
– iTunes will not open if you have notification off.
– If you use .lock and do not specify lock or unlock, it toggles.
– Fixed a bug with the D flag.
– Added TRIVIA!
– Fixed a user monitor bug.
– Made commands process faster.
– The back command now goes back instead of forward.

Luxer Update! Beta #5

– If a user with A is unbanned, they will be automatically banned.
