Luxer beta 14 released

Leave a reply

I released beta 14 of Luxer today.

Changes:
– Added a greet option.
– Made help links to news and forums work.
– Tweaked anti-flood (will not flood out on .help anymore)
– Added set command… .help set for more information.

You can download the new version here: http://marcusfamily.info/~ryan/forums/index.php/topic,15.0.html

I’ve discovered a security hole in Luxer. Anybody can use any custom command, regardless of access. Anybody with an access of 1 can use ALL custom commands.

Last updated by Ryan Marcus at July 29, 2006.
