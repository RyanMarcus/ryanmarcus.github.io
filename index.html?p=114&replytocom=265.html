Luxer beta 11 released

3 Replies

I released beta 11 of Luxer today.

Changes:
– The say command now works correctly.
– The user database is no longer case sensitive.
– Added connect on disconnect
– Added connect on open
– Added a VERY beta chat interface
– Interface redone by Frank

To get the new version, look here: http://marcusfamily.info/~ryan/forums/index.php/topic,15.new.html
