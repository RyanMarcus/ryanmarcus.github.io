Luxer beta 10 is underway.

Leave a reply

I finally found the time to work on Luxer, and beta 10 is going to be great!

I completely redesigned the user interface, moving from a “one large window” approach to a “many small window” approach. This means you can pick exactly what you want to have open and what you don’t want. No more clunky tabs, either.

I’m also converting all of my evil text file preferences and storage into databases… So you won’t see users load one by one when you open the user window.

The flag system is also going to be completely redone… Instead of having every single letter be a flag, you will be able to assign an action to any letter. If you want B to be autoban, then you can set it that way. If you would rather have B declare the user as a bot, you can do that as well!

And lastly, and most importantly (in my opinion), I’m redoing the whole java base connection! No more 50/50 chance of connecting!

I wish I could offer you a solid release date, but I can’t. Anywhere between today and two weeks from now, just keep checking this blog or opening Luxer (it checks for updates when it starts up.)
