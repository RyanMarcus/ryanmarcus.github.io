Luxer Version 1.0 Update

15 Replies

I’ve done a bit of work on all 9 of the features that I’ve promised for Luxer 1.0.

  • Easy install. Download and double click, as it should be.
    I’ve figured out how to run the shell install script from a REALbasic application. Should not be long now!
  • More iTunes commands
    I’m planning on adding the following iTunes features:

    1. PlayRating RATING(1-5): Skips forward until a song of RATING is found.
    2. Notification: Sends you a whisper with song information as the song changes
    3. openitunes and quititunes: Commands to open and quit iTunes
  • Remember CDKeys for different clients
    Finished.
  • Phrasebans
    I’ve finished this feature 100%. You can look at the documentation forum and check it out.
  • Vote
    Have not done much on this yet, still brainstorming about the best way to do it.
  • Chat window
    Almost complete. A few bugs to work out, and we got it!
  • Be able to change the access level required for commands
    Have not even touched it. This one will be a pain.
  • Make custom commands easier
    Done! You can now double click a command in the custom commands window to edit it, which is faster then remaking the command everytime…
  • Implement Mail
    This was a lot easier then it sounded. I created a mail database, and you can use the mail and inbox commands just like most other bots. Any person can have an unlimited number of messages sent to them, and when they join the channel they will be notified of their new mail.

I’ve also added a few extra goodies:

  • Faster connection: I optimized the code Luxer uses to connect. It will be faster.
  • APM: You can now use the commands apm START apm STOP and apm RESULTS to see and measure your actions per minute.

    • Demember 14th, here we come!
